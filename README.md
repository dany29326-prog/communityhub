# CommunityHub Landing Page

A premium, modern SaaS landing page designed to showcase **CommunityHub** — a real, fully-functional community management platform built for Homeowners Associations (HOAs), condominiums, residential communities, and membership organizations.

## Tech Stack
* **Markup:** HTML5
* **Styling:** Tailwind CSS (via CDN)
* **Icons:** Font Awesome Icons (via CDN)
* **Fonts:** Inter (Google Fonts)
* **Interactions:** Vanilla JavaScript (Lightbox modal, smooth anchor scroll, responsive design)

## Features
* **Premium SaaS Aesthetic:** Dark-mode themed layout inspired by Stripe, Linear, and Framer, featuring subtle glowing borders and backdrop blur cards.
* **Interactive Showcases:** Responsive CSS device frames (laptops & phones) featuring real product screenshots.
* **Built-in Lightbox:** Seamless JavaScript lightbox to inspect product screenshots in high resolution.
* **Optimized UX:** Smooth anchor scrolling, direct email/LinkedIn contact cards for the founder, and fully responsive layout.

## Live Demo
You can view the project locally by opening `index.html` in any web browser, or host it on **GitHub Pages**.
